# 异步编程

const user = await getUser()
const product = await getProduct()
1
2

getUser()和getProduct()没有同步关系，但是需要这两者都 resolve 才继续执行

这种情况可以考虑使用 Promise.all

const [user, product] = await Promise.all([getUser(), getProduct()])
1

但是使用Promise.all的潜在问题是，当其中一个promise的状态是reject时，其他promise也会失败。可以使用Promise.allSettled()替代

当然也可以不立即使用await

const user = getUser()
const product = getProduct()

// here are some code...

await user
await product
