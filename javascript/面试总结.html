# 重绘和重排（回流）有什么区别

  • 重排 布局属性（大小，尺寸、位置等等）发生变换，重新计算该属性，然后重新描绘该元素
  • 重绘 可见性（颜色等等）属性发生变化，重新绘制元素
  • 重排必定导致重绘

# 伪类和伪元素

伪类和伪元素是为了修饰不在文档树中的部分 伪类用于 已有元素的处于某种状态时(hover...) 伪元素可以创建一些不在文档树里面的元素

# object 和 map 的区别

  • map 的键可以是 Object 类型
  • object 的键不能被迭代只能通过 for in 访问
  • map 的插入顺序遵从一定的规则

# JavaScript 浮点数

JavaScript的浮点数是64位浮点数，遵从 IEEE 754 标准， 在一些精度要求比较高的场景下，需要使用 定点数 替代 浮点数

# Symbol 的作用

Symbol 是为了提供唯一不可变的值，用作非字符串属性，可以防止属性冲突。 Symbol 创建的属性和私有属性有点类似，但不是私有属性，私有属性可以由闭包产生

# 基本的数据类型

Javascript里面分为简单类型和复杂类型，简单类型有以下几种

  • undefined
  • null
  • symbol
  • bigint (新增)
  • boolean
  • number
  • string

还有一种复杂的类型是 object

可以使用 typeof（常用于基本类型的判断） 以及 instanceof 进行类型判断，特殊情况是 typeof null 返回的是 object
